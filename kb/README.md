# Aspen Wallet Knowledge Base

This `/kb` directory contains developer-facing documentation for the Aspen Wallet WordPress plugin.

## Documents

- [`architecture.md`](./architecture.md): system architecture, module map, and execution flow.
- [`data-model.md`](./data-model.md): wallet storage model, meta keys, and balance semantics.
- [`integrations.md`](./integrations.md): WooCommerce, Subscriptions, Fluent Booking, and FluentCRM integration details.
- [`shortcodes.md`](./shortcodes.md): shortcode contracts, sanitization, and rendering behavior.
- [`operations.md`](./operations.md): setup, manual QA checklist, troubleshooting, and extension notes.

## Audience

These docs are written for:

- plugin developers extending functionality,
- QA engineers validating wallet flows,
- maintainers troubleshooting production behavior.
